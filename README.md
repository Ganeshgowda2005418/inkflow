# 🚀 InkFlow

A modern blog platform built with Node.js, Express.js, and EJS that allows users to create, edit, and manage blog posts through a clean and responsive interface.

---

## ✨ Features

* 📝 Create blog posts
* ✏️ Edit existing posts
* 🗑️ Delete posts
* 📱 Responsive design
* ⚡ Dynamic EJS templating
* 🔄 Express.js routing
* 🎨 Clean and modern UI/UX
* 📲 Mobile-friendly layout

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* EJS
* HTML5
* CSS3
* JavaScript

